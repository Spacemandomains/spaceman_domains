# Wilfred L Lee Jr - Portfolio Website

## Overview
Static portfolio website for Wilfred L Lee Jr, AI Founder & Senior UX Designer. Pre-built React app served via a simple Node.js static file server.

## Project Architecture
- `index.html` - Main entry point
- `assets/` - Pre-built JS, CSS, and image assets
- `serve.js` - Simple Node.js static file server (port 5000)
- `favicon.png` - Site favicon
- `opengraph.jpg` - Open Graph social sharing image

## Running
The app runs via `node serve.js` on port 5000, serving static files.
